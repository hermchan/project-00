#Racing Game

A simple two player racing game. Who can button smash harder?

## Technologies Used**

* HTML
* jQuery
* CSS
* Javascript
* Object constructor

## Existing Features

* Event handlers that detect and move players
* Winner declared with an alert
* Reset button
* Customized background and player icons


## Planned Features

* Include a countdown timer
* Scorekeeper
